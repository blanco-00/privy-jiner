# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-03-13

### Added

- Initial open source release
- Core framework:
  - Agent coordination system
  - Task management
  - Memory management
  - Event bus
  - Configuration management
  - Logging system
  - SQLite database with WAL mode
  - Plugin interface and loading
- Standalone deployment mode:
  - WebSocket server
  - HTTP API server
  - Session management
- AI integration:
  - Model configuration
  - OpenAI client
  - Claude client
  - Token tracking
- Modules:
  - Finance management
  - Health tracking
  - Fashion/wardrobe
  - Knowledge base
  - News aggregation
- Internationalization:
  - Chinese (zh) support
  - English (en) support
  - Language detection
- Data privacy:
  - Local storage
  - AES-256 encryption
  - Backup/restore
  - Export (JSON/CSV)
- Database abstraction:
  - SQLite support
  - MySQL support
  - PostgreSQL support

### Known Issues

- Some modules still need test data
- OpenClaw plugin integration needs testing
- UI dashboard is minimal
