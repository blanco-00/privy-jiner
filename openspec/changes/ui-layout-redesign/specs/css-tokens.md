# UI Design System Specification

## Color Palette

### Primary Colors
- `--bg-primary`: #0f0f0f (main background)
- `--bg-secondary`: #1a1a1a (cards)
- `--bg-tertiary`: #242424 (inputs, hover)

### Accent Colors
- `--accent-primary`: #e8a854 (gold - primary actions)
- `--accent-success`: #54e88a (green)
- `--accent-danger`: #e85454 (red)
- `--accent-info`: #54a8e8 (blue)

### Text Colors
- `--text-primary`: #f5f5f5
- `--text-secondary`: #888888
- `--text-tertiary`: #666666

### Border Colors
- `--border-primary`: #333333
- `--border-hover`: #444444
